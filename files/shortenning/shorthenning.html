Скорочення

border- : width => style => color; (Можна задавати у будь-якому порядку і лише ті, які потрібно.)

background- : color => image => repeat; (Наприклад: background: white url(images/cocktail.gif) repeat-x;

font- : style (курсив): italic/normal => weight (жирність): bold/normal => size (розмір шрифту): px/pt/% => height (міжрядковий інтервал): px => family (сімейство шрифту) Приклад: font:italic bold 12px/20px arial;

margin- : top => right => bottom => left;

padding- : top => right => bottom => left;
