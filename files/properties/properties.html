background-color

- колір фону: 

 body{
   background-color: #3366CC; /* Цвет фона веб-страницы */
 } 
 h1{
   background-color: RGB(249, 201, 16); /* Цвет фона под заголовком */
 }

background-image - малюнок для фону елементу. Але для логтипу краще використовувати елемент img background-image:url(images/1.png);

background-position - розташування фонового малюнку: http://htmlbook.ru/css/background-position

background-repeat: no-repeat; - щоб не повторювався малюнок фону http://htmlbook.ru/css/background-repeat

background-position - розташування малюнку фону: http://htmlbook.ru/css/background-position

border (px) - обведення тексту.

border-bottom/top/right/left - розмежувальна лінія: http://htmlbook.ru/css/border-bottom

border-top/bottom/right/left-color коли потрібно задати колір одної лінії

border-color - колір розмежувальної лінії

border-width - ширина розмежувальної лінії

border-style - стиль розмежувальної лінії http://htmlbook.ru/css/border-style

box-shadow - тінь: https://www.w3schools.com/cssref/css3_pr_box-shadow.asp

clear - забороняє плаваючому елементу знаходитися всередині http://htmlbook.ru/css/clear

box-sizing - як рахувати контент: http://code.mu/ru/markup/manual/css/property/box-sizing/

cursor - http://htmlbook.ru/css/cursor

display block - елемент веде себе, як блочний, а inline - рядковий, inline-block - разом. Більше інформації: https://www.youtube.com/watch?v=za_2ogi312Y&feature=emb_logo http://htmlbook.ru/css/display

display:flex; - елемент стає, як би гумовим, який розтягується. Яким саме чином розтягується, дивись тут: https://www.w3schools.com/css/css3_flexbox_container.asp

float - вирівнювання по сторінці: http://htmlbook.ru/css/float

font-weight (lighter/normal/bold/bolder) - жирність.

font-style (italic/normal/oblique) - курсив.

font-size (px/pt/%/em) - розмір тексту: https://www.w3schools.com/cssref/pr_font_font-size.asp. Примітка: краще використовувати для body розміри словами, а решту у відсотках, або em.

font-family - тип шрифту. http://old.code.mu/books/css/osnovnye-css-svojstva-dlya-novichkov.html
Існує 5 типів шрифту, які необхідно вказувати у кінці списку: serif, sans-serif, cursive, fantasy та monospace.

height (px/%) - висота змісту.

letter-spacing - інтервал мід символами

line-height (px/em/%/число) - міжрядковий інтервал. Якщо задати просто число, це значитиме, що розмір інтервалу залежатиме від самого шрифту. https://puzzleweb.ru/css/pr_line-height.php

list-style - стиль списку: http://htmlbook.ru/css/list-style

list-style-position - коли потрібно, щоб не виходили за рамку позначки списку (необхідне значення inside) http://code.mu/ru/markup/manual/css/property/box-sizing/

margin-top/bottom/left/right - відступ. Зручніше його використовувати, ніж line-height: http://htmlbook.ru/faq/kak-izmenit-rasstoyanie-mezhdu-abzatsami-teksta (можна задавати значення auto)

outline - замість margin, але не займає місця: http://code.mu/ru/markup/manual/css/property/outline/

padding (-left/right/top/bottom) - відстань між змістом (текстом) та границею (border) елементу (значення auto не існує, на відміну від margin)

position (static/relative/fixed/absolute/sticky) - https://www.w3schools.com/css/css_positioning.asp

text-align (left/right/center/justify) - вирівнювання змісту (тексту). Працює лише із блочними елементами. Його не можна застосувати до, наприклад, елементу img окремо, лише, якщо цей елемент буде всередині блочного елементу.

text-decoration (none/underline/overline/line-through)

text-indent (відступ першого рядка абзацу): px

text-transform - http://htmlbook.ru/css/text-transform

vertical-align (лише для таблиць): bottom/middle/top

width (px/pt) - ширина змісту.

z-pozition - http://htmlbook.ru/css/z-index
