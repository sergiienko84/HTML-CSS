Різниця між класом та ідентифікатором: 

class - для багатьох елементів. 
    HTML: class="n";
    CSS: .n - для будь-якого елементу, p.n - лише для абзаців.
id - для унікального елементу.
    HTML: id="n"
    CSS: #n - для будь-якого елементу, p#n - лише для абзаців.

div - елементу можна привласнити як класс class, так і ідентифікатор id (Дивись вище)
Створення нащадків: 


    div h2{
      color: black;        
    }
    #elixir h2{
      color: black;
    }
  div p b{
      color: black;        
    }

p.test - такой селектор выбирает абзацы с классом test. А вот если я сделаю так: p .test (поставлю пробел), то выберу все элементы с классом test, находящиеся внутри абзацев. Детальніше тут: http://old.code.mu/books/css/osnovy-raboty-s-selektorami-css-dlya-novichkov.html

transition: color 0.2s linear; - плавно міняє колір. Наприклад: 


.nav__link{
  color:#fff;
  text-decoration: none;
  transition: color 0.2s linear;
}
.nav__link:hover{
  color: #fce38a;
}
